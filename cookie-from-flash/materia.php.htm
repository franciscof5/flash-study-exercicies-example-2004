Abrir PopUp no Flash com JS e ASP.
por Thiago Della Serra

Aqui há uma demonstração da interação Flash / JavaScript / ASP, onde através de um botão no flash é aberta uma nova janela (popup) que chama um arquivo .asp. Neste arquivo pode haver uma validação, uma contagem, etc.

Insira no HEAD na página que contém seu movie a função Javascript, como no exemplo abaixo.

<HTML>
<HEAD><TITLE>Flash e Javascript - www.asparena.eti.br/flash </TITLE>
<script language="JavaScript">
<!--
function AbreJanela(arquivo, tamx, tamy)
//onde arquivo é seu documento.asp, tamx é o tamanho horizontal em pixels e tamy tamanho vertical em pixels
{
window.open(Foto,"","resizable=no,toolbar=no,status=no,menubar=no,scrollbarsno,width=" + tamx + ",height=" + tamy)
}
// -->
</script>
</HEAD>
<BODY>

...
aqui vêm as tags do Flash
...

</BODY>
</HTML>


e no flash basta criar um botão com o seguinte ActionScript:

On (Release)
   Get URL ("javascript:AbreJanela('nome_do_arquivo.asp',300,150)")
End On

onde 300, 150 determinarão as dimensões da janela.
