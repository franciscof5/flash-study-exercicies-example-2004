Executar funções de JavaScript (Flash+JavaScript).
por Ricardo Figueira

Objectivo :

Como associar funções feitas em JavaScript no flash. Vamos analizar um pouco de como interagir essas duas tecnologias.

Executar funções :

Suponhamos que temos uma função em JavaScript chamada "NovaJanela()", e que efectua a abertura de uma nova janela no Browser com o conteúdo da página "janela.htm" com as seguintes dimensões (width=550,height=400):

<script language="JavaScript">
<!--
function NovaJanela()
{
window.open("janela.htm","Image3","toolbar=no,location=no, directories=no,status=no, menubar=no,scrollbars=no,resizable=no,width=550,height=413");
}
//-->
</script>

No flash executamos esta função atravéz de um botão na qual se deverá colocar nas actions o seguinte:

On (Release)
   Get URL ("JavaScript:NovaJanela()")
End On

Executamos esta função atravéz da instrução "Get URL" sendo apenas necessário escrever JavaScript:Nomedafunção().

Teremos que ter em atenção que esta função (JavaScript) deverá estar incluida no ficheiro (html) onde consta a abertura do movie em Flash.

Aprendemos então a maneira de como executar funções em JavaScript no Flash sendo apenas o limite ..... a vossa imaginação.

Espero que vos tenha sido util :)
